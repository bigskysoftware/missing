Link

A link widget provides an interactive reference to a resource. The target resource can be either external or local, i.e., either outside or within the current page or application.

Notes

Missing.css uses the <a> element for links.

When an <a> tag is used to link to content on the same page, missing.css will provide the following visual context:

Headings
A :before pseudo-element whose color is determined by the --accent CSS variable.
Other
An outline of the element whose color is determined by the --fg CSS variable.

Keyboard Interaction

  • Enter: Executes the link and moves focus to the link target (when focused).
