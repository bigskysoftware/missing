Tabs

Tabs are a set of layered sections of content, known as tab panels, that display one panel of content at a time. Each tab panel has an associated tab element, that when activated, displays the panel. The list of tab elements is arranged along one edge of the currently displayed panel, most commonly the top edge.

Notes

Missing.css provides the <aria-tablist>, <aria-tab>, and <aria-tabpanel> custom elements for tabs.

  • Don't forget to set an accessible label for the <aria-tablist>.
  • You must establish the relationship between <aria-tab> and <aria-tabpanel> elements by providing [aria-controls] attributes to each <aria-tab>. The component will set the reverse [aria-labelledby] attributes (generating unique ids if necessary).
  • You must set the initial state with <aria-tab aria-selected="true"> and <aria-tabpanel hidden>).

Keyboard Interaction

  • Tab: Tab in and out of the tablist, remembering previous focus.
  • Shift+Tab: Tab in and out of the tablist, remembering previous focus.
  • Left Arrow: Move focus to the previous tab.
  • Right Arrow: Move focus to the next tab.
  • Home: Move focus to the first tab.
  • End: Move focus to the last tab.
